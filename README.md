# Enterprise CRM System

A complete CRM project for a web development course. It includes lead tracking, customer accounts, sales pipeline, dashboard analytics, activity logs, and role-based access control.

## Features

- Login system with Admin and Sales roles
- Lead tracking with status updates
- Customer account management
- Sales pipeline with deal stages
- Dashboard with revenue, pipeline value, lead sources, and deal stage analytics
- Activity logs for important actions
- Admin-only lead delete permission
- Data saved in data/db.json
- Responsive design for laptop and mobile screens

## How To Run

1. Install Node.js from https://nodejs.org
2. Open this project folder in VS Code
3. Open the terminal
4. Run: npm start
5. Open: http://localhost:3000
